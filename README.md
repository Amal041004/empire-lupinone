# Empire: LupinOne VulnHub Penetration Test

## 📌 Overview
This repository documents the complete penetration testing lifecycle of the 
Empire: LupinOne virtual machine from VulnHub. The assessment was conducted 
in a controlled virtual lab environment using Kali Linux as the attacker system.

---

## 🎯 Objective
- Perform reconnaissance and enumeration
- Identify vulnerabilities
- Exploit misconfigurations
- Escalate privileges
- Obtain root access

---

## 🖥️ Lab Setup

| Role     | Machine |
|----------|----------|
| Attacker | Kali Linux |
| Target   | Empire: LupinOne |
| Network  | 192.168.150.0/24 (Host-Only) |

---

## 🔄 Methodology Workflow

1. Network Verification
2. Host Discovery
3. Port & Service Enumeration
4. Web Enumeration
5. Directory Fuzzing
6. Key Extraction & Decoding
7. SSH Access
8. Horizontal Privilege Escalation
9. Vertical Privilege Escalation
10. Root Flag Capture

---

## 🛠 Tools Used

- Nmap
- FFUF
- CyberChef
- John The Ripper
- LinPEAS
- Python Module Hijacking
- GTFOBins

---

## ✅ Final Result

✔ User access obtained  
✔ Privilege escalation successful  
✔ Root access achieved
